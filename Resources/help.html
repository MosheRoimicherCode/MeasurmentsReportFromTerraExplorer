Terra Explorer Add-On Help

Tool Overview

This tool allows you to create a report based on 3D objects within Terra Explorer. You can generate points, lines, and polygons and organize them into groups. Afterward, you can export the data, along with snapshots of the objects, to a PDF.

How to Use

Creating Groups and Objects:

Create new groups inside Terra Explorer using the tool. Add points, lines, or polygons within the group. You can also create and modify other elements using Terra Explorer’s GUI.
Note: The tool only allows creation of objects. For style modifications (e.g., color or properties), right-click on the object and select "Properties."

Naming Objects and Groups:

Once created, you can assign names to both the group and individual objects.

Generating a Report:

Press the Report button to generate the PDF report. Choose the language of the PDF output (English or Hebrew). Select the folder and PDF file name. After clicking, the tool will take snapshots of the 3D objects and lock the user interface until the PDF is ready.

Completion:

Once the PDF is generated, the application will unlock and open the PDF automatically.

סקירה כללית של הכלי

כלי זה מאפשר לך ליצור דוח המבוסס על אובייקטים תלת-ממדיים בתוך Terra Explorer. ניתן ליצור נקודות, קווים ופוליגונים ולארגן אותם בקבוצות. לאחר מכן, ניתן לייצא את הנתונים, יחד עם תמונות של האובייקטים, לקובץ PDF.

כיצד להשתמש:

יצירת קבוצות ואובייקטים:

צור קבוצות חדשות בתוך Terra Explorer באמצעות הכלי. הוסף נקודות, קווים או פוליגונים בתוך הקבוצה. ניתן גם ליצור ולשנות אובייקטים נוספים באמצעות הממשק הגרפי של Terra Explorer.
הערה: הכלי מאפשר רק יצירת אובייקטים. לשינויים בסגנון (כגון צבע או מאפיינים), לחץ לחיצה ימנית על האובייקט ובחר "מאפיינים".

מתן שמות לאובייקטים ולקבוצות:

לפני יצירתם, ניתן להקצות שמות גם לקבוצה וגם לאובייקטים הבודדים.

אופצה נוספת, שינוי שם דרך עץ השכבות של התוכנה.

יצירת דוח:

לחץ על כפתור הדוח ליצירת דוח PDF. בחר את שפת הדוח (עברית או אנגלית). בחר את התיקייה ושם קובץ ה-PDF. לאחר הלחיצה, הכלי יצלם תמונות של האובייקטים התלת-ממדיים וינעל את ממשק המשתמש עד שה-PDF יהיה מוכן.

השלמה:

לאחר יצירת ה-PDF, היישום ייפתח מחדש וה-PDF ייפתח אוטומטית.
